# TASK-003: Скачивание видео из YouTube, RuTube, VK Video

## Статус

Backlog

## Описание

Добавить возможность скачивания видео по ссылкам из YouTube, RuTube и VK Video. Бот определяет доступные разрешения (144p–720p), предлагает пользователю выбрать качество через инлайн-клавиатуру, скачивает и отправляет видео в чат. Для поддержки файлов до 2GB поднимается Local Telegram Bot API сервер в Docker-контейнере.

## Приоритет

Medium
