# Руководство пользователя

Полная документация по использованию Claude Code Telegram Bot.

---

## Что умеет бот

- **Текст** — отправь любой вопрос или задачу, Claude ответит
- **Голос** — отправь голосовое сообщение, бот распознает речь и ответит текстом + голосом
- **Фото** — отправь скриншот или фото документа, бот извлечёт текст (OCR) и обработает
- **Документы** — отправь файл, бот прочитает содержимое и передаст Claude
- **Файлы от Claude** — скриншоты, CSV и другие файлы, созданные Claude, автоматически отправляются в чат
- **Длинные ответы** — вместо спама из 10+ сообщений: первый чанк + `.md` файл с полным ответом

## Команды

| Команда | Описание | Кто может |
|---------|----------|-----------|
| `/start` | Приветствие и информация | Все |
| `/help` | Справка и список команд | Все |
| `/new` | Новая сессия (сброс контекста) | user, admin |
| `/cancel` | Отменить текущий запрос | user, admin |
| `/model` | Сменить модель (haiku / sonnet / opus) | user, admin |
| `/model <имя>` | Переключиться на конкретную модель | user, admin |
| `/voice` | Включить/выключить голосовые ответы | user, admin |
| `/status` | Текущее состояние (модель, сессия, голос) | user, admin |
| `/usage` | Твоя статистика за сегодня | user, admin |
| `/stats` | Статистика всех пользователей | admin |

## Типы ввода

### Текстовые сообщения

Просто напиши вопрос или задачу в чат. Claude обработает и ответит.

Примеры:
- «напиши функцию сортировки на Python»
- «объясни что делает этот код»
- «исправь баг в файле main.py»

### Голосовые сообщения

Отправь голосовое сообщение — бот распознает речь (faster-whisper), передаст текст Claude и ответит:
- Текстом в чат
- Голосовым сообщением (если включён голосовой режим через `/voice`)

Цепочка обработки: 🎤 OGG → ffmpeg → WAV → faster-whisper → текст → Claude → ответ → edge-tts → голос

### Фотографии

Отправь фото с подписью или без:
- Бот извлечёт текст через OCR (tesseract)
- Если есть подпись — использует как задачу
- Передаст всё Claude для обработки

Сценарии:
- Скриншот ошибки + «исправь» → анализ и решение
- Фото документа + «извлеки текст» → распознанный текст
- Код с экрана + «что делает этот код» → объяснение

### Документы

Отправь файл (`.py`, `.txt`, `.json`, `.csv` и др.) — бот прочитает содержимое и передаст Claude.

## Навигация и bash

Claude понимает bash-команды прямо в чате:

```
ls                              — список файлов
cd my-project                   — перейти в проект
mkdir new-project && cd new-project — создать и перейти
pwd                             — текущая директория
git init / git status / ...     — git команды
```

Или на естественном языке:
- «создай папку my-app и перейди в неё»
- «покажи содержимое текущей директории»
- «найди все Python файлы в проекте»

## Модели

Переключай модель командой `/model <имя>`:

| Модель | Особенности | Когда использовать |
|--------|------------|-------------------|
| `haiku` | Быстрая, дешёвая | Простые вопросы, мелкие задачи |
| `sonnet` | Баланс скорости и качества | По умолчанию. Большинство задач |
| `opus` | Максимальное качество | Сложные задачи, архитектура, анализ |

## Роли и лимиты

| Роль | Что может | Лимиты |
|------|-----------|--------|
| `admin` | Всё + `/stats` | Без ограничений |
| `user` | Текст, голос, фото, документы | Дневной лимит сообщений |
| `readonly` | Только чтение ответов | Дневной лимит сообщений |

Текущий лимит можно посмотреть командой `/usage`.

## Сессии и контекст

- Claude помнит контекст разговора в рамках сессии
- `/new` — сброс сессии (начать с чистого листа)
- `/status` — показать ID текущей сессии

Когда сбрасывать сессию:
- Переключаешься на другую тему
- Контекст стал слишком большим
- Claude «запутался» в предыдущих сообщениях

## FAQ

**Бот не отвечает?**
Подожди — Claude может думать 10-60 секунд. Если ответа нет больше 2 минут, используй `/cancel` и попробуй снова.

**Как сбросить контекст?**
Команда `/new`.

**Как сменить модель?**
Команда `/model opus` (или `sonnet`, `haiku`).

**Голосовые ответы не приходят?**
Включи голосовой режим: `/voice`.

**Фото не обрабатываются?**
На сервере должен быть установлен tesseract-ocr. Обратись к администратору.

**Ответ слишком длинный?**
Бот автоматически отправит `.md` файл с полным текстом, а в чат — краткую версию.

**Что значит лимит сообщений?**
Администратор может ограничить количество сообщений в день. Проверь свой лимит: `/usage`.
