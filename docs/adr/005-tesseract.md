# ADR-005: Tesseract для OCR

**Статус:** Принято
**Дата:** 2026-03-06

## Контекст

Бот принимает фотографии из Telegram. Claude Code CLI не поддерживает vision (передачу изображений). Нужен способ извлечь текст из фото и передать его Claude как текстовый промпт.

## Решение

pytesseract 0.3.13 + системный пакет `tesseract-ocr` с языковым пакетом `tesseract-ocr-rus`.

Ключевые аргументы:
- **Локальный** — не требует внешнего API, работает на CPU.
- **Мультиязычный** — `lang="rus+eng"` распознаёт смешанный русско-английский текст.
- **Ленивая проверка** — доступность tesseract проверяется один раз при первом фото.
- **Стандарт** — самый распространённый OCR, установлен через `apt install tesseract-ocr`.

Ограничение: распознаёт только текст. Для визуального контента (диаграммы, скриншоты UI) — недостаточно.

## Альтернативы

- **EasyOCR** — лучше качество на сложных изображениях, но тяжелее (~1 ГБ моделей), медленнее на CPU.
- **Google Cloud Vision API** — высокое качество, но платный и внешний сервис.
- **PaddleOCR** — хорошее качество, но сложнее установка и зависимости.
