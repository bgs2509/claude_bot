# ADR-003: faster-whisper для STT

**Статус:** Принято
**Дата:** 2026-03-06

## Контекст

Бот принимает голосовые сообщения из Telegram (формат .ogg). Нужна транскрибация речи в текст (STT) с поддержкой русского языка. Сервер может не иметь GPU.

## Решение

faster-whisper 1.1.0 с моделью `base` и квантизацией `int8`.

Ключевые аргументы:
- **Локальный** — не требует внешнего API, работает на CPU. Данные не покидают сервер.
- **CTranslate2** — в 4 раза быстрее оригинального Whisper на CPU при том же качестве.
- **Ленивая загрузка** — модель загружается при первом голосовом сообщении, экономя ~500 МБ RAM до этого.
- **Конфигурируемый** — `WHISPER_MODEL` (tiny/base/small) и `WHISPER_DEVICE` (cpu/cuda) через `.env`.

## Альтернативы

- **OpenAI Whisper** — оригинальная реализация, медленнее в 4 раза на CPU, больше потребление памяти.
- **OpenAI Whisper API** — облачный, требует API-ключ и оплату, данные уходят во внешний сервис.
- **Google Speech-to-Text API** — высокое качество, но платный и требует Google Cloud аккаунт.
- **Vosk** — лёгкий, но хуже качество распознавания русского языка.
