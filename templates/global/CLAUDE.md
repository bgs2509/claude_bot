# Глобальные настройки Claude Code

## Язык
Всегда отвечай на русском языке.

## Предпочтения
- Предпочитай простые решения сложным
- Не создавай лишних файлов, папок, структур без необходимости

## Безопасность
- НЕ читать .env, *.pem, *.key, credentials — только .env.example
- НЕ изменять файлы вне текущего проекта ($PWD), кроме ~/ai-steward/**/*.md
- НЕ логировать пароли, токены, ключи
- Enforce: см. ~/.claude/settings.json (deny rules, sandbox)

## Инструментарий
- При наличии подходящего skill — использовать его
- MCP-серверы: использовать для GitHub, Docker, браузер
- Agent teams: использовать для параллельных задач
- Плагины: python-pipeline для ревью и тестов Python-кода
- Skills имеют приоритет над ручным выполнением аналогичной задачи

## Адаптация к пользователю
- Определяй уровень пользователя по контексту проекта
- Dev-проекты — технический язык
- Life-проекты — простой, человеческий язык без жаргона
- Не предполагай техническую грамотность если не указано в профиле

## Автообновление CLAUDE.md
При изменениях — обновляй CLAUDE.md на соответствующем уровне иерархии.
Приоритет: реальное состояние > CLAUDE.md. При конфликте — обновить CLAUDE.md.

## Мульти-машинность
Этот файл должен быть идентичным на всех машинах.
Машино-специфичное — в ~/.claude/settings.local.json.
